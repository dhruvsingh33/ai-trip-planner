AI Trip Planner is a modern travel planning application where users can:
- Sign in using Google
- Create trips by selecting destination, days, and preferences
- Get AI-generated itineraries
- View places and hotel recommendations
- Save and manage trips securely

This project is designed for **learning, demos, and resume showcase**.
